Hello from Group 7! This is our Docker + Jenkins + Terraform pipeline!

This project is to showcase the our automation skills! We choose to do this by showing integration between Docker, Jenkins, and our new technology: Terraform! We are able to monitor any changes with Jenkins with webhooks, use Docker to build the website application, and use Terraform to deploy the updated app.

The Problem:

Manual deployments of webapplications can be slow and inconsistent to deploy. You'd need to manually rebuild the Docker container and redeploy each time to see the code changes you made.

The Solution:

Use Jenkins to automatically remake the Docker image and Terraform build and redeploy the Docker container it for you! If it's approved on github then the code automatically goes live on the webpage, with Jenkins monitoring the changes!

This is something easy to change to show proof!
